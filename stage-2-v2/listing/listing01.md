Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:

```text
77 78 79
слайсинг [1:4] делает выбор включая элемент с индексом 1(77) и не включая элемент с индексом 4(80)
математическая запись [1:4)
```
